# My Website

Website built with [Docusaurus](https://docusaurus.io/).

## Installation

```bash
npm install
```

## Commands

Here are the most common commands you will need:

### Start Development Server

```bash
npm run start
```

This command starts a local development server and serves your site on `http://localhost:3000/`. Most changes are reflected live without having to restart the server.

### Build Website

```bash
npm run build
```

This command generates static content into the `build` directory. You can host this directory on any static hosting service.

### Deploy

```bash
npm run deploy
```

This command builds the site and pushes it to GitHub Pages (if configured in `docusaurus.config.js`).

## Your Directory Structure

- `blog/`: Contains the blog posts.
- `docs/`: Contains the markdown files for your documentation.
- `src/`: Contains custom CSS and React components.
- `static/`: Contains static assets like images and logos.
- `docusaurus.config.js`: Contains the site configuration.
- `package.json`: Contains project dependencies.
- `sidebars.js`: Contains the order of documents in your sidebar.
